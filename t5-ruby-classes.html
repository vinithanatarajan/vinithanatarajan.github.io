Creating a Class in Ruby

Classes represent a type of an object, such as a pen, a book, a flower, or lemon.Classes are created via the class keyword. After that comes the name of the class. All class names must start with a Capital Letter. By convention, we use CamelCase for class name. So we would create classes like PieceOfCake, but not like Piece_of_Cake.

Let's make a Lemon Class:

class Lemon

def Lemon.color

return "yellow"

end

def self.size

return "kinda small"

end

class << self

def shape

return "lemon-ish"

end

end

end

Lemon.color # => "yellow"

Lemon.size # => "kinda small"

Lemon.shape # => "lemon-ish"

The next section defines a class method. A class method is a method that is defined for a particular class. In the below example we call the Lemon.color method of the Lemon Class.

outside of a method definition in a class block, self refers to the class itself. It is preferred, as it makes changing the name of the class much easier. ClassName.method_name and self.method_name are essentially the same.
