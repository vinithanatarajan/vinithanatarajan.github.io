Arrays and Hashes in Ruby

Arrays: Arrays are ordered, integer-indexed collections of any object. Arrays are used to store collections of data. Each object in an array has an unique key assigned to it. We can access any object in the array using this unique key. The positions in an array starts from " 0 ". The first element is located at " 0 ", the second at 1st position etc.

Example: Try the following in - irb.

TV = Array.new

TV = %w[Samsung, Panasonic, LG, Sony]

we have added 4 elements in the array.

puts TV[2]

LG,

Add a new element to position 4.

TV[4] = "Sansui"

Hashes: Like arrays, Hashes are also used to store data. Hashes points an object to another object. Consider of assigning a certain "meaning" to a string.Each time we refer that string, it refers its "meaning". Hashes have a default value that is returned when accessing keys that do not exist in the hash. By default, that value is nil.

Example:

TV = Hash.new

TV = { 'Samsung' => 'Samsung LED 46", Samsung LED 55", Samsung LED 60" and Samsung LED 65"',

'Panasonic' => 'Panasonic LED 46", Panasonic LED 55" and Panasonic LED 60"',

'LG' => 'LG LED 46", LG LED 55", and LG LED 60"' }

Try this now:

TV['Samsung']

You get => ""Samsung LED 46", Samsung LED 55", Samsung LED 60" and Samsung LED 65""
