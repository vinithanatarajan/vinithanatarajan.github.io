Inline vs Inline blocks

  • Inline elements:
    • It has left & right margins and padding, but not top & bottom.
    • It cannot have a width and height set.
    • It allows other elements to sit to their left and right.

      • An inline element has no line break before or after it.

  • Inline-block elements:
    • It allows other elements to sit to their left and right.
    • It has top & bottom margins and padding.
    • It has height and width

      • An inline-block element is placed as an inline element (on the same line as adjacent content), but it behaves as a block element.

Example:

While using display: inline; the two paragraph flow together as a single paragraph.

While using display: inline block; the two paragraphs appear right next to each other.
